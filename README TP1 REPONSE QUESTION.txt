 Le message du fichier chiffrer: boYboY1/1boYboY1;
 
 On peut voir que le message certes avec des lettre différente ressemble a mon message “coucou ? coucou !”
 
 On peut remarque a vue d'oeil que :

Lettre originale | Lettre chiffrer

      b          =       c
      o          =       o
      Y          =       u
      1          =      " "
      /          =       ?
      ;          =       !
      
On peut supposer que  signifie le end line.
